Kick command

The Kick command is a moderation command. Use this command to kick people from your server! The kick command of Cheeztos is activated by "/kick user:<@user>", This kick command is quite effective. Although if you have a large server, You might want to use the ban command since kicking is not very effective against the same account.

Ban command

The ban command is a moderation command. Use this command to ban people from your server permanently, When a user gets banned, He is banned forever from that account and IP address. ALthough this might prove to be ineffective against VPN alts that join from another IP, But this command can ensure you a permanent ban of the main account.

Mute command

The mute command is quite effective against people who constantly annoy you, Raid your server. This mute command can be quite effective against them, Use "/mute <@user> to mute the user temporarily. This command will make the user unable to talk in any channels."
